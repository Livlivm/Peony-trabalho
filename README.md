# Peony Ateliê

## Descrição
O Peony Ateliê é um aplicativo desenvolvido com o objetivo de proporcionar uma experiência de compra confortável, prática e personalizada para os clientes.

A plataforma oferece diversas opções de produtos e permite que o usuário crie sua própria peça personalizada, tornando cada compra única.

## Proposta
Nosso projeto foi pensado para melhorar a experiência do usuário durante compras online, oferecendo:

- Variedade de produtos  
- Facilidade de navegação  
- Personalização de peças  
- Interface intuitiva e agradável

##  Como Executar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/peony-atelie.git
```

2. Acesse a pasta do projeto:
```bash
cd peony-atelie
```

3. Instale as dependências:
```bash
npm install
```

4. Execute o projeto:
```bash
npm start
```

---

## Como Usar

1. Acesse a aplicação  
2. Navegue pelos produtos disponíveis  
3. Escolha um item ou personalize sua própria peça  
4. Finalize a experiência de compra
